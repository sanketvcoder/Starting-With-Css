    • Simple Web Page

      There are three basic ways to add CSS

      1. inline css
      2. Inline CSS

        3. Inline CSS is used to apply a unique style to a single HTML element. It uses the style attribute of the HTML element.

      3. internal css
      4. Internal CSS

        5. Internal CSS is used to define styles in the head section of an HTML page, within a <style> tag. This is useful when a single document has a unique style.

      5. external css
      6. External CSS

        7. External CSS is used to define styles for multiple HTML pages. With an external stylesheet, you can change the look of an entire website by changing just one file.

      Apply Inline css Apply Internal css Apply External css

                                        inlineSubDiv.style.display = 'block';
                                    }

                                    function applyInternalCss() {
                                        contentDiv.removeAttribute('style');
                                        document.querySelector('#internalCss').disabled = false;

                                        hideAllSubDivs();
                                        internalSubDiv.style.display = 'block';

                                    // Function to apply External CSS and show sub-div
                                    function applyExternalCss() {
                                        contentDiv.removeAttribute('style');
                                        document.querySelector('#internalCss').disabled = true;
                                        document.querySelector('#externalCss').disabled = false;
                                        hideAllSubDivs();
                                        externalSubDiv.style.display = 'block';
                                    }
                        
                                    // Event listeners for buttons
                                    inlineBtn.addEventListener('click', applyInlineCss);
                                    internalBtn.addEventListener('click', applyInternalCss);
                                    externalBtn.addEventListener('click', applyExternalCss);
