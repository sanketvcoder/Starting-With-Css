There are three basic ways to add CSS

  1. inline css
  2. Inline CSS

    3. Inline CSS is used to apply a unique style to a single HTML element. It uses the style attribute of the HTML element.

  3. internal css
  4. Internal CSS

    5. Internal CSS is used to define styles in the head section of an HTML page, within a <style> tag. This is useful when a single document has a unique style.

  5. external css
  6. External CSS

    7. External CSS is used to define styles for multiple HTML pages. With an external stylesheet, you can change the look of an entire website by changing just one file.
